# 📊 Sales Dashboard (Tableau)

## 📌 Project Overview
This project presents an interactive **Sales Dashboard** built using Tableau to analyze business performance across different dimensions like revenue, profit, and product sales.

The dashboard helps in identifying key trends, top-performing products, and overall business insights in a clear and visual manner.

---

## 🎯 Objectives
- Track overall sales performance  
- Analyze profit and revenue trends  
- Identify top-performing products  
- Provide clear and interactive visual insights  

---

## 📁 Dataset Details
The dataset includes:
- Order Date  
- Product Name  
- Sales (Revenue)  
- Profit  
- Quantity  
- Category / Sub-category  

---

## 📊 Dashboard Features

### 🔹 KPI Cards
- Total Revenue  
- Total Profit  
- Total Orders  

### 🔹 Sales Trend
- Line chart showing sales over time  

### 🔹 Profit Analysis
- Profit distribution across categories  

### 🔹 🏆 Top Products (Bar Chart)
- Displays **top-performing products based on sales**
- Helps quickly identify best-selling items  

### 🔹 Filters
- Date filter  
- Category filter  
- Interactive selection options  

---
